Css Tutorials

Cascading style sheets(css) is used to format the Html of the website to make it beautiful and attractive.

Adding Css To HTML Page

There are three ways to add css to html page.

  • Inline Css
  • Internal Css
  • External Css
Cascading order defines which style will be applied to element when multiple styles are used.
Cascading order priority: Inline > ( internal == external ) > browser default.

The internal and external CSS are treated equally by the browser. But the order of their definition decides which one's property will get preference.
If link to external CSS is defined before the internal CSS, then properties of internal CSS will get the preference, i.e internal CSS > external CSS.
If link to external CSS is defined after the internal CSS, then properties of external CSS will get preference over internal CSS, i.e. external CSS > internal CSS.
=> Inline styles
Inline styles are used using the style attribute along with the element tag. Example:
<p style="color: red;">Inline Css</p>
=> Internal styles
Internal styles are used using the style element in the head section of the html page. Example:
<style>
p { color: red; }
</style>
=> External styles
External styles are applied using a complete different CSS file. To use external stylesheet, a link tag is used in the head section of the html page.
<link rel="stylesheet" href="style.css" type="text/css">
The rel attribute defines the relationship between the html page and the linked file.
The href attribute defines the path of the linked file.
The type attribute defines the type of the linked file.

Css Selectors

There are mainly three types of selectors in css.

  • Element Selector
  • Class Selector
  • Id Selector
  • pseudo classes
=> Element Selector
Element selector is used to select all the elements of the same type. Example:
p { color: red; }
=> Class Selector
Class selector is used to select all the elements of the same class. we just have to write a dot(.) before the class name. Example:
for html element: <p class="red">Class Selector</p>
appying css like this : .red { color: red; }
=> Id Selector
Id selector is used to select a single element of the same id. we have to write a hash(#) before the id name. There could be only one id in the tag, and the id is unique in the html page. Example:
for html element: <p id="red">Id Selector</p>
appying css like this : #red { color: red; }
=> Grouping Selectors
Grouping selectors are used to apply same css to multiple selectors. we have to write a comma(,) between the selectors. Example:
p, .red, #code { color: red; }
=> Nesting Selectors
Nesting selectors are used to apply css to the child elements of a parent element. we have to write a space( ) between the selectors, such that the sequence reperesents a hierarchy, startin from the top. Example:
p .red { color: red; }
we can refer to the direct child element of a parent element by using the greater than sign(>) between the selectors. Example:
p > .red { color: red; }
=> chaining selectors
chaining selectors are used to apply css to the element tags having a specific class or id. only the element tags having the specific class or id will be selected. Example:
p.red { color: red; }
p#code { font-family: 'Lucida Console' }
=> pseudo classes
pseudo classes are used to apply css to the elements in a specific state. Example:
a:hover { color: red; }

section:nth-child(2n) { background-color: blue; } //every even numbered section will have background color as blue

section:nth-child(2n+1) { background-color: green; } //every odd numbered section will have background color as green

p::first-line { font-weight: bold; } //the first line of every paragraph will be bold.

p::first-letter { font-size: 2em; } //the first letter of every paragraph will be 2 times bigger than the rest of the letters.

//to get different images as list-style-type, we can use the following code:
ul { list-style-type: none; }
li::before {
content: '';
background-image: url('image.png');
display: inline-block;
height: 16px;
width: 16px;
background-size: contain;
}

CSS Background

The background properties are used to set the background effects of an element.

=> background-color
The background-color property is used to set the background color of an element. Example:
p { background-color: red; }
=> background-image
The background-image property is used to set the background image of an element. by default , a background image is placed at the top left corner of an element, and is repeated both vertically and horizontally.
The values that it can take are:
  • url(uniform resource link) - specifies the url of the image. we can specify more than one image by separating them with a comma(,). Example:
    p { background-image: url('image.jpg'); }
  • linear-gradient - sets a linear gradient as the background image. at least two colors needed to be mentioned. the direction of the gradient can be specified by mentioning the angle in degrees or by mentioning the direction as to right top, to right bottom, to left top, to left bottom. Example:
    p { background-image: linear-gradient(to right, red, blue); }
  • radial-gradient - sets a radial gradient as the background image. at least two colors needed to be mentioned. Example:
    p { background-image: radial-gradient(red, blue); }
  • repeating-linear-gradient - sets a repeating linear gradient as the background image. at least two colors needed to be mentioned. Example:
    p { background-image: repeating-linear-gradient(red, blue); }
  • repeating-radial-gradient - sets a repeating radial gradient as the background image. at least two colors needed to be mentioned. Example:
    p { background-image: repeating-radial-gradient(red, blue); }
=> background-repeat
The background-repeat property is used to set the background image repeatation of an element. Example:
p { background-repeat: repeat-x; }
The background-repeat property can have following values:
  • repeat - the background image will be repeated both horizontally and vertically, this is default.
  • repeat-x - the background image will be repeated horizontally.
  • repeat-y - the background image will be repeated vertically.
  • no-repeat - the background image will not be repeated.
  • round - the image is repeated and shrink or stretched to fit the element.
=> background-position
The background-position property is used to set the background image position of an element. by default , the image is placed at the top left corner of an element. the value can be of the form 'x y'. x represents the horizontal position and y represents the vertical position. it can take values in pixels(px), percentages(%), or keywords like left, right, top, bottom, center. Example:
p { background-position: left top; }
=> background-attachment
The background-attachment property is used to specify whether a background image should scroll or be fixed. Example:
p { background-attachment: fixed; }
The background-attachment property can have following values:
  • scroll - this is default value. it will scroll with page.
  • fixed - the background image will not scroll with the page.
  • local - the background image will with the element's contents.
=> background-size
The background-size property is used to specify the size of the background image. Example:
p { background-size: 100px 100px; }
it can take values in pixels(px), percentages(%), or keywords like cover, contain, auto.
  • auto - this is default value. the background image will be displayed in its original size.
  • length or percentage - the first value is width and second value is height.
  • cover - the background image will be resized to cover the entire width of the element.
  • contain - the background image will be resized to fit the entire element.

Css Units

There are two types of units in css:

  • Absolute Units - these units are fixed and do not change with the size of the browser window.
    • px - pixels
    • cm - centimeters
    • mm - millimeters
    • in - inches
    • pt - points
    • pc - picas

  • Relative Units - these units are relative to other length units.
    • % - relative to parent element.
    • em - relative to the font-size of the element
    • rem - relative to font-size of the root element
    • vw - relative to 1% of the width of the viewport
    • vh - relative to 1% of the height of the viewport

CSS Border

The CSS border properties allow you to specify the style, width, and color of an element's border.
border is shorthand property for border-width, border-style, and border-color.
Example:

p { border: 1px solid black; }
The different properties of border are:
=> border-style
The border-style property is used to specify the style of the border.
Example:
p { border-style: solid; }
The border-style property can have following values:
  • none - no border
  • hidden - same as none, but takes space
  • dotted - a dotted border
  • dashed - a dashed border
  • solid - a solid border
  • double - a double border
  • groove - a 3D grooved border. The effect depends on the border-color value
  • ridge - a 3D ridged border. The effect depends on the border-color value
  • inset - a 3D inset border. The effect depends on the border-color value
  • outset - a 3D outset border. The effect depends on the border-color value
  • inherit - inherits this property from its parent element.
=> border-width
The border-width property is used to specify the width of the border.
Example:
p { border-width: 5px; }
The border-width property can have following values:
  • thin - a thin border
  • medium - a medium border
  • thick - a thick border
  • length - specifies the width in px, pt, cm, etc.
  • initial - sets this property to its default value.
  • inherit - inherits this property from its parent element.
=> border-color
The border-color property is used to specify the color of the border.
Example:
p { border-color: red; }
we can provide different colors to different sides of border.
Example:
p { border-color: red green blue yellow; }
=> border individual sides
we can also provide different border to different sides of an element.
Example:
p { border-top: 1px solid black; }
p { border-bottom: 1px solid black; }
p { border-left: 1px solid black; }
p { border-right: 1px solid black; }
=> border-radius
The border-radius property is used to add rounded borders to an element.
Example:
p { border-radius: 5px; }
we can also provide different radius to different sides of an element.
Example:
p { border-top-left-radius: 5px; }
p { border-top-right-radius: 5px; }
p { border-bottom-left-radius: 5px; }
p { border-bottom-right-radius: 5px; }

Here is an interesting example :

#border{
border-style: dotted;
border-radius: 10px 40px;
border-color: rgb(26, 11, 27);
background-color: rgb(115, 91, 223);
width: 40%;
border-width: 10px 4px;
text-align: center;
}

Text and font styling

Font properties define the look of the font like font family, boldness, size, style etc.
Text properties define the layout and presentation of text, like text alignment, text decoration, and text transformation.

=> font-size
This defines the size of the font. it can be absolute , or relative.
Example:
p { font-size: 20px; }
=> font-family
This defines the font family of the text. it should hold multiple values,
in case the first font is not available, the second one will be used, and so on.
if the name of a font family is more than one word, it must be in quotation marks, like: "Times New Roman".
Example:
p { font-family: "Times New Roman", Times, serif; }
=> font-style
This defines the style of the font. it can be normal, italic, oblique, initial, inherit.
Example:
p.normal { font-style: normal; }
p.italic { font-style: italic; }
p.oblique { font-style: oblique; }
=> font-weight
This defines the weight of the font. it can be normal, bold, bolder, lighter, initial, inherit. it can also be a number from 100 to 900. Example:
p.bold { font-weight: bold; }
p.number { font-weight: 230; }
=> font-variant
This defines whether the font should be displayed in small capital letters.
Example:
p.normal { font-variant: normal; }
p.small { font-variant: small-caps; }
=> text-align
This defines the horizontal alignment of the text. it can be left, right, center, justify, initial, inherit. Example:
p.left { text-align: left; }
p.right { text-align: right; }
p.center { text-align: center; }
p.justify { text-align: justify; }
=> text-decoration
This defines the decoration of the text. it can be none, underline, overline, line-through, initial, inherit. Example:
p.none { text-decoration: none; }
p.underline { text-decoration: underline; }
p.overline { text-decoration: overline; }
p.line-through { text-decoration: line-through; }
=> text-transform
This defines the capitalization of the text. it can take the following values:
  • uppercase - turns every character to uppercase.
  • lowercase - turns every character to lowercase.
  • capitalize - turns the first character of every word to uppercase.
  • none - text renders as it is . it is default.
Example:
p.capitalize { text-transform: capitalize; }
#upper { text-transform: uppercase; }
=> text-indent
This defines the indentation of the first line in a text-block.
Example:

This adds shadow to the text. it can take 3 values: which are position of the shadow on the x-axis, position of the shadow on the y-axis, and the color of the shadow. Example:

=> line-height
This defines the height of a line. it can be a absolute, or a percentage.
Example:
p { line-height: 1.5px; }

This defines the spacing between letters. it can be a absolute, or a percentage.

Margins and Padding

=> The margin property is used to create space between the borders and the surrounding elements. it can have values as absolute or percentage.
or 'auto' which will apply on the left and right margins of a block element, and center it.
and we can provide margins separately to the sides by using 'margin-top', 'margin-bottom', 'margin-left', 'margin-right'. example:

p { margin: 10px; } //all sides 10px
p { margin: 10px 20px; } //top and bottom 10px, left and right 20px
p { margin: 10px 20px 30px; } //top 10px, left and right 20px, bottom 30px
p { margin: 10px 20px 30px 40px; } //top 10px, right 20px, bottom 30px, left 40px

=> The padding property is used to create space between the content and the border. it can have values as absolute or percentage.
and we can provide padding separately to the sides by using 'padding-top', 'padding-bottom', 'padding-left', 'padding-right'. example:
p { padding: 10px; } //all sides 10px
p { padding: 10px 20px; } //top and bottom 10px, left and right 20px
p { padding: 10px 20px 30px; } //top 10px, left and right 20px, bottom 30px
p { padding-top: 10px; } //top 10px

Css Position

the position property is for specifying the positioning of element on the page relative to web page or parent element. it has following values: static, relative, absolute, fixed, sticky.
but this property is not enough to manipulate the position of element, therefore, to move the elements we use these 4 properties: left, right, top, bottom.
the above properties i.e. left,right, top , bottom have numerical values in both relative and absolute units.

Lets explain the position property one by one :

=> static
this is the default property. properties like left, right , top, bottom does not work when this is used.
=> relative
the relative positions the element relative to its first positioned ancestor element. and now the properties like left, right , top, bottom works.
the original position of the element remains occupied. example:
html code :

<div class="outer">
 <h2>outer box</h2>
 <div class="inner"> inner box relative to the outer div </div>
 <div> this element is placed with its upper space as blank, which shows that the space is occupied by the inner box </div>
</div>

css code :

.outer{
 width: 60%;
 height: 100px;
 border: 1px solid black;
 background-color: lightpink;
 margin: 0px auto;
 padding: 5px;
}

.inner{
 position: relative;
 top: 40px;
 left: 30px;
 width: 70%;
 border: 1px solid black;
 background-color: lightgreen;
}
will show:
outer box
inner box relative to outer box
this element is placed with its upper space as blank, which shows that the space is occupied by the inner box.

-> the inner box is 40px from top and 30px from left relative to its original position.

=> absolute
the absolute positions the element relative to its immediate parent element. the parent element should be relative. if there is no relative immediate parent element then the element will be positioned relative to the body element. and the properties like left, right , top, bottom works.
the original position of the element does not remains occupied. example:
html code :

<div class="outer_absolute">
 <span>outer box</span>
 <div class="inner_absolute"> inner box absolute to the outer div </div>
 <div> this element is placed just after the text 'outer box' text, which shows that the space is not occupied by the inner box </div>
</div>

css code :

.outer_absolute{
 width: 60%;
 height: 100px;
 border: 1px solid black;
 background-color: lightpink;
 margin: 0px auto;
 padding: 5px;
 position: relative;
}

.inner_absolute{
 position: absolute;
 top: 40px;
 left: 30px;
 width: 70%;
 border: 1px solid black;
 background-color: lightgreen;
}
will show:
outer box
inner box absolute to outer box
this element is placed just after the text 'outer box' text, which shows that the space is not occupied by the inner box.

-> the inner box is 40px from top and 30px from left relative to its parent(outer box).

=> fixed
the fixed positions the element relative to the viewport, which means it always stays in the same place even if the page is scrolled. and the properties like left, right , top, bottom works.
the original position of the element does not remains occupied.
example: the 'go to top button' on the bottom right side of the page.
html code :

<a href="#"><img id="back" src="../images/back_to_top.png" alt="backtotop"></a> css code :

#back{
 position: fixed;
 bottom: 8px;
 right: 8px;
 border-radius: 50%;
 width: 30px;
 height: 30px;
}
=> sticky
A sticky element toggles between relative and fixed, depending on the scroll position. it is positioned relative until a given position is met in the viewport , then it sticks in place ( like fixed). example:
html code :

<div class="outer_sticky">
 <span>outer box</span>
 <div class="inner_sticky"> inner box sticky to the outer div </div>
 <div> this element is placed just after the text 'outer box' text, which shows that the space is occupied by the inner box </div>
</div>

 top: 300px;

outer box

Media Queries

It is used to make the website more responsive. The order of media queries matters, the last media query that is satisfying will be applied.
its syntax is :

@media mediatype and (media feature) {
 css-code;
}
example:
#div1{
 width: 60%; //default width
 border : 1px solid black;
}
@media screen and (max-width: 600px) { //if screen size is less than 600px
 #div1 { width:80%; }
 #main { margin-left : 5px; }
}
@media screen and (max-width: 400px) { //if screen size is less than 400px
 #div1 { width:100%; }
 #main { margin-left : 2px; }
}

//here the last media query that is satisfying will be applied.
so, if the screen size is greater than 600px, then the default width of div1 will be 60%. and if the screen size is less than 600px but greater than 400px, then the width of div1 will be 80%. and if the screen size is less than 400px, then the width of div1 will be 100%.


=> To make the website more responsive, we set the width of the elements in percentage with max-width and min-width. example:
.outer{
 width: 80%;
 min-width: 300px;
 max-width: 600px;
}

Css Transition

The transition property is a shorthand property for the four transition properties:

  • transition-property : specifies the name of the CSS property the transition effect is for.
  • transition-duration : specifies how many seconds or milliseconds a transition effect takes to complete.
  • transition-timing-function : specifies the speed curve of the transition effect. it can have the following values:
    • ease : specifies a transition effect with a slow start, then fast, then end slowly (this is default).
    • linear : specifies a transition effect with the same speed from start to end.
    • ease-in : specifies a transition effect with a slow start.
    • ease-out : specifies a transition effect with a slow end.
    • ease-in-out : specifies a transition effect with a slow start and end.
    • cubic-bezier(n,n,n,n) : lets you define your own values in a cubic-bezier function.
  • transition-delay : specifies a delay (in seconds) for the transition effect.
example:
html code :

<div id="backtransition">
 this box will change its color when hovered
</div>

css code :

#backtransition{
 width: 200px;
 height: 100px;
 border: 1px solid black;
 background-color: blue;
 transition: background-color 1s ease-in-out;
}

#backtransition:hover{
 background-color: red;
}
will show:
this box will change its color when hovered

Css Transform

=> The transform property can transform an element in 2D or 3D space.
it can have the following values:

  • rotate(n deg) : rotates an element clockwise from its current position.
  • scale(n,n) : increases or decreases the size of an element.
  • translate(x,y) : moves an element from its current position (according to the parameters given for the X-axis and the Y-axis).
  • skew(x-angle,y-angle) : skews an element along the X and Y-axis by the given angles.
  • matrix(n,n,n,n,n,n) : combines all the 2D transform methods into one.
example:
html code :

<div id="transform">
 this box will rotate when hovered
</div>

css code :

#transform{
 width: 200px;
 height: 100px;
 border: 1px solid black;
 background-color: blue;
 transition: transform 1s ease-in-out;
}

#transform:hover{
 transform: rotate(180deg);

Css animation

=> The animation property is a shorthand property for the following animation properties:

  • animation-name : specifies the name of the @keyframes animation.
  • animation-duration : specifies how many seconds or milliseconds an animation takes to complete one cycle.
  • animation-timing-function : specifies the speed curve of the animation. it can have the following values:
    • ease : specifies an animation with a slow start, then fast, then end slowly (this is default).
    • linear : specifies an animation with the same speed from start to end.
    • ease-in : specifies an animation with a slow start.
    • ease-out : specifies an animation with a slow end.
    • ease-in-out : specifies an animation with a slow start and end.
    • cubic-bezier(n,n,n,n) : lets you define your own values in a cubic-bezier function.
  • animation-delay : specifies a delay (in seconds) for the animation to start.
  • animation-iteration-count : specifies the number of times an animation should be played. it can have the following values:
    • infinite : specifies that the animation should be played infinite times (this is default).
    • number : specifies the number of times an animation should be played.
  • animation-direction : specifies whether or not the animation should play in reverse on alternate cycles. it can have the following values:
    • normal : specifies that the animation should be played as normal (this is default).
    • reverse : specifies that the animation should be played in reverse direction.
    • alternate : specifies that the animation should be played forwards first, then backwards.
    • alternate-reverse : specifies that the animation should be played backwards first, then forwards.
