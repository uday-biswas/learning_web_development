Css Tutorials

Cascading style sheets(css) is used to format the Html of the website to make it beautiful and attractive.

Adding Css To HTML Page

There are three ways to add css to html page.

  • Inline Css
  • Internal Css
  • External Css
Cascading order defines which style will be applied to element when multiple styles are used.
Cascading order priority: Inline > ( internal == external ) > browser default.

The internal and external CSS are treated equally by the browser. But the order of their definition decides which one's property will get preference.
If link to external CSS is defined before the internal CSS, then properties of internal CSS will get the preference, i.e internal CSS > external CSS.
If link to external CSS is defined after the internal CSS, then properties of external CSS will get preference over internal CSS, i.e. external CSS > internal CSS.
=> Inline styles
Inline styles are used using the style attribute along with the element tag. Example:
<p style="color: red;">Inline Css</p>
=> Internal styles
Internal styles are used using the style element in the head section of the html page. Example:
<style>
p { color: red; }
</style>
=> External styles
External styles are applied using a complete different CSS file. To use external stylesheet, a link tag is used in the head section of the html page.
<link rel="stylesheet" href="style.css" type="text/css">
The rel attribute defines the relationship between the html page and the linked file.
The href attribute defines the path of the linked file.
The type attribute defines the type of the linked file.

Css Selectors

There are mainly three types of selectors in css.

  • Element Selector
  • Class Selector
  • Id Selector
=> Element Selector
Element selector is used to select all the elements of the same type. Example:
p { color: red; }
=> Class Selector
Class selector is used to select all the elements of the same class. we just have to write a dot(.) before the class name. Example:
for html element: <p class="red">Class Selector</p>
appying css like this : .red { color: red; }
=> Id Selector
Id selector is used to select a single element of the same id. we have to write a hash(#) before the id name. There could be only one id in the tag, and the id is unique in the html page. Example:
for html element: <p id="red">Id Selector</p>
appying css like this : #red { color: red; }
=> Grouping Selectors
Grouping selectors are used to apply same css to multiple selectors. we have to write a comma(,) between the selectors. Example:
p, .red, #code { color: red; }
=> Nesting Selectors
Nesting selectors are used to apply css to the child elements of a parent element. we have to write a space( ) between the selectors, such that the sequence reperesents a hierarchy, startin from the top. Example:
p .red { color: red; }
we can refer to the direct child element of a parent element by using the greater than sign(>) between the selectors. Example:
p > .red { color: red; }
=> chaining selectors
chaining selectors are used to apply css to the element tags having a specific class or id. only the element tags having the specific class or id will be selected. Example:
p.red { color: red; }
p#code { font-family: 'Lucida Console' }

CSS Background

The background properties are used to set the background effects of an element.

=> background-color
The background-color property is used to set the background color of an element. Example:
p { background-color: red; }
=> background-image
The background-image property is used to set the background image of an element. by default , a background image is placed at the top left corner of an element, and is repeated both vertically and horizontally.
The values that it can take are:
  • url(uniform resource link) - specifies the url of the image. we can specify more than one image by separating them with a comma(,). Example:
    p { background-image: url('image.jpg'); }
  • linear-gradient - sets a linear gradient as the background image. at least two colors needed to be mentioned. the direction of the gradient can be specified by mentioning the angle in degrees or by mentioning the direction as to right top, to right bottom, to left top, to left bottom. Example:
    p { background-image: linear-gradient(to right, red, blue); }
  • radial-gradient - sets a radial gradient as the background image. at least two colors needed to be mentioned. Example:
    p { background-image: radial-gradient(red, blue); }
  • repeating-linear-gradient - sets a repeating linear gradient as the background image. at least two colors needed to be mentioned. Example:
    p { background-image: repeating-linear-gradient(red, blue); }
  • repeating-radial-gradient - sets a repeating radial gradient as the background image. at least two colors needed to be mentioned. Example:
    p { background-image: repeating-radial-gradient(red, blue); }
=> background-repeat
The background-repeat property is used to set the background image repeatation of an element. Example:
p { background-repeat: repeat-x; }
The background-repeat property can have following values:
  • repeat - the background image will be repeated both horizontally and vertically, this is default.
  • repeat-x - the background image will be repeated horizontally.
  • repeat-y - the background image will be repeated vertically.
  • no-repeat - the background image will not be repeated.
  • round - the image is repeated and shrink or stretched to fit the element.
=> background-position
The background-position property is used to set the background image position of an element. by default , the image is placed at the top left corner of an element. the value can be of the form 'x y'. x represents the horizontal position and y represents the vertical position. it can take values in pixels(px), percentages(%), or keywords like left, right, top, bottom, center. Example:
p { background-position: left top; }
=> background-attachment
The background-attachment property is used to specify whether a background image should scroll or be fixed. Example:
p { background-attachment: fixed; }
The background-attachment property can have following values:
  • scroll - this is default value. it will scroll with page.
  • fixed - the background image will not scroll with the page.
  • local - the background image will with the element's contents.
=> background-size
The background-size property is used to specify the size of the background image. Example:
p { background-size: 100px 100px; }
it can take values in pixels(px), percentages(%), or keywords like cover, contain, auto.
  • auto - this is default value. the background image will be displayed in its original size.
  • length or percentage - the first value is width and second value is height.
  • cover - the background image will be resized to cover the entire width of the element.
  • contain - the background image will be resized to fit the entire element.

Css Units

There are two types of units in css:

  • Absolute Units - these units are fixed and do not change with the size of the browser window.
    • px - pixels
    • cm - centimeters
    • mm - millimeters
    • in - inches
    • pt - points
    • pc - picas

  • Relative Units - these units are relative to other length units.
    • % - relative to parent element.
    • em - relative to the font-size of the element
    • rem - relative to font-size of the root element
    • vw - relative to 1% of the width of the viewport
    • vh - relative to 1% of the height of the viewport

CSS Border

The CSS border properties allow you to specify the style, width, and color of an element's border.
border is shorthand property for border-width, border-style, and border-color.
Example:

p { border: 1px solid black; }
The different properties of border are:
=> border-style
The border-style property is used to specify the style of the border.
Example:
p { border-style: solid; }
The border-style property can have following values:
  • none - no border
  • hidden - same as none, but takes space
  • dotted - a dotted border
  • dashed - a dashed border
  • solid - a solid border
  • double - a double border
  • groove - a 3D grooved border. The effect depends on the border-color value
  • ridge - a 3D ridged border. The effect depends on the border-color value
  • inset - a 3D inset border. The effect depends on the border-color value
  • outset - a 3D outset border. The effect depends on the border-color value
  • inherit - inherits this property from its parent element.
=> border-width
The border-width property is used to specify the width of the border.
Example:
p { border-width: 5px; }
The border-width property can have following values:
  • thin - a thin border
  • medium - a medium border
  • thick - a thick border
  • length - specifies the width in px, pt, cm, etc.
  • initial - sets this property to its default value.
  • inherit - inherits this property from its parent element.
=> border-color
The border-color property is used to specify the color of the border.
Example:
p { border-color: red; }
we can provide different colors to different sides of border.
Example:
p { border-color: red green blue yellow; }
=> border individual sides
we can also provide different border to different sides of an element.
Example:
p { border-top: 1px solid black; }
p { border-bottom: 1px solid black; }
p { border-left: 1px solid black; }
p { border-right: 1px solid black; }
=> border-radius
The border-radius property is used to add rounded borders to an element.
Example:
p { border-radius: 5px; }
we can also provide different radius to different sides of an element.
Example:
p { border-top-left-radius: 5px; }
p { border-top-right-radius: 5px; }
p { border-bottom-left-radius: 5px; }
p { border-bottom-right-radius: 5px; }

Here is an interesting example :

#border{
border-style: dotted;
border-radius: 10px 40px;
border-color: rgb(26, 11, 27);
background-color: rgb(115, 91, 223);
width: 40%;
border-width: 10px 4px;
text-align: center;
}
