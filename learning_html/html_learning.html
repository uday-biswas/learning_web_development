html tags and attributes

=>The tag name can either start from an alphabet or an underscore( _ ).
=> Attributes can provide additional information about the HTML elements on your page and control their behaviour.
Eg: <tag_name attribute_name="value"> Content Enclosed </tag_name>
=>HTML tags are not case sensitive, <p> means the same as <P>

html headings

=>there are 6 heading tags from <h1> to <h6> , the size and boldness of the text inside the header tags will go on decreasing as we go from h1 to h6.
=>we can use this header tags multiple times in a single html file.
=>the user agent css style is : 

        h1 {
            display: block;
            font-size: 2em;
            margin-block-start: 0.67em;
            margin-block-end: 0.67em;
            margin-inline-start: 0px;
            margin-inline-end: 0px;
            font-weight: bold;
        }
=> eg. <h1> heading 1 </h1>

html paragraphs

=>paragraphs are defined with the <p> tag, and is most commonly used for writing paragraphs
=>the user agent css style is : 

        p {
            display: block;
            margin-block-start: 1em;
            margin-block-end: 1em;
            margin-inline-start: 0px;
            margin-inline-end: 0px;
        }
=>eg. <p> paragraph </p>

=> the contents inside <pre> and </pre> tags are shown as it is, without any formatting.
usually it reserves the space and the line breaks.
the user agent css style is: 
        pre {
            display: block;
            font-family: monospace;
            white-space: pre;
            margin: 1em 0px;
        }
=> the <br> tag is used to insert a single line break.
=> the <hr> tag is used to insert a horizontal line below. the user agent css style is: 
 
    hr {
        display: block;
        unicode-bidi: isolate;
        margin-block-start: 0.5em;
        margin-block-end: 0.5em;
        margin-inline-start: auto;
        margin-inline-end: auto;
        overflow: hidden;
        border-style: inset;
        border-width: 1px;
    }
=> the <abbr> tag is used to insert an abbreviation.
      eg. <abbr title="World Health Organization">WHO</abbr> ->WHO
=> the <address> tag is used to insert an address.
      eg. <address> 123, abc street, xyz city, 123456 </address> ->
