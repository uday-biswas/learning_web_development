html text formatting

=> the <abbr> tag is used to insert an abbreviation.
      eg. <abbr title="World Health Organization">WHO</abbr> ->WHO
=> the <address> tag is used to define the contact information of the author or owner of the document or article.
      eg. <address> 123, abc street, xyz city, 123456 </address> ->
123, abc street, xyz city, 123456
=> the <b> and <strong> tags are used to make bold text.
      eg. <b> bold or strong text </b> -> bold or strong text
=> the <bdo> tag is used to insert a text with a different direction.
      eg. <bdo dir="rtl"> text with a different direction </bdo> -> text with a different direction
=> the <blockquote> tag is used to insert a block of quoted text.
      eg. <blockquote> quoted text </blockquote> ->
quoted text
=> the <cite> tag is used to insert a citation.
      eg. <cite> citation </cite> -> citation
=> the <code> , <kbd> , <samp> tags are used to make a text look like written by typewritter.
      eg. <code> this is the way </code> -> this is the way
=> the <del> and <s> tags are used to make a deleted or strikethrough text.
      eg. <del> deleted or strikethrough text </del> -> deleted or strikethrough text
=> the <dfn> tag is used to insert a definition.
      eg. <dfn> definition </dfn> -> definition
=> the <em> and <i> tag is used to insert emphasized or italic text.
      eg. <em> emphasized or italic text </em> -> emphasized or italic text
=> the <ins> and <u> tags are used to insert a underlined or inserted text.
      eg. <u> underlined or inserted text </u> -> underlined or inserted text
=> the <mark> tag is used to insert a marked text.
      eg. <mark> marked text </mark> -> marked text
=> the <pre> tag is used to insert a preformatted text, the spaces and line breaks are kept as it is.
      eg. <pre> preformatted text </pre> ->
 preformatted text
=> the <small> tag is used to make small sized text.
      eg. <small> small text </small> -> small text

      eg. (a + b)<sub> 2 </sub> ->(a + b)2
=> the <sup> tag is used to insert a superscript text.
      eg. (a + b)<sup> 2 </sup> ->(a + b)2
=> the <var> tag is used to insert a variable.
      eg. <var> variable </var> -> variable
=> the <wbr> tag is used to insert a possible line-break.
      eg. <wbr> possible line-break </wbr> -> possible line-break

html links

=> the <a> tag is used to insert a link.
      eg. <a href="http://www.google.com"> google </a> -> google
=> By default, links will appear as follows in all browsers:
>An unvisited link is underlined and blue
>A visited link is underlined and purple
>An active link is underlined and red
=> By default, the linked page will be displayed in the current browser window.
Use the attribute "target" to open the linked document in a new browser window or tab:
Example: <a href="https://www.w3schools.com/" target="_blank">Visit W3Schools!</a> -> Visit-W3Schools!
=> to use image as link , we just have to put the img tag inside the <a> tag.
<a href="../learning_html/login_page.html"> <img src="../images/icons8-gunter-100.png" alt="penguin photo" target="_blank" style="width:42px;height:42px;"> </a> ⇛
=>Use mailto: inside the href attribute to create a link that opens the user's email program (to let them send a new email):
Example: <a href="mailto:udaybiswasofficial@gmail.com"> email </a> ->Send-email-to-us.
=> if the page is too long , then to access different parts of the page :
First, use the id attribute to create a bookmark:
<h3 id="quote">html quotation</h3>
Then, add a link to the bookmark using a "#" before that id:
Example : <a href="#quote">html quotation</a> -> html quotation
You can also add a link to a bookmark on another page:
<a href="static_page.html#bold1"> players list </a> -> players list
=> the <link> tag is used to link external files.
eg. <link rel="stylesheet" href="mystyle.css" type="text/css"> // here link tag is used to link with the external css file
eg. <link rel="icon" type="image/icon" href="../image/favicon.jpg"> //here link tag is used to add the favicon to the webpage
=> the <nav> tag is used to define navigation links.
eg. <nav> navigation links </nav> ->
navigation links

html images

=> The HTML tag is used to embed an image in a web page.
the essential attributes of the tag are: src, alt, float, height and width.
src - defines the path to the image , alt - defines an alternate text for the image, and the css style attribute is used to define the float, width and height of an image.
the width is given in percentage and the height is given in pixels to get the the best result.
Example : <img src="../images/icons8-gunter-100.png" alt="penguin photo" style="float:right;width:42px;height:42px;">



html image maps

=> An image map is used to make an image with clickable areas.
The HTML <map> tag is used to define an image map, and then it is linked with the <img> tag using "usemap" attribute.
The <area> tag defines an area inside an image map, it is present inside <map> tag, and it has several attributes such as : href, shape, coords, alt, target, download.
=> shape can be circle, rect, poly, default.
>for circle, the coordinates are x,y,radius.(x,y are coordinates of center of the circle)
>for rect(rectangle), the coordinates are x1,y1,x2,y2.(x1,y1 are coordinates of top-left corner of the rectangle and x2,y2 are coordinates of bottom-right corner of the rectangle)
>for poly(polygon), the coordinates are x1,y1,x2,y2,x3,y3,....,xn,yn.(x1,y1 are coordinates of first point of the polygon and so on)
>for default, the coordinates are not required, it selects the entire region.
Example : <img src="../images/workplace.jpg" style="float:left;margin-right:5px;" usemap="#image-map">
<map name="image-map">
<area target="_blank" alt="tea-cup" href="https://www.cup.com/" coords="337,302,46" shape="circle">
<area target="_blank" alt="laptop" title="hp laptops" href="https://www.hp.com/" coords="31,40,274,348" shape="rect">
<area target="_blank" alt="mobile" href="https://www.samsung.com/" coords="289,172,327,172,334,245,293,244" shape="poly">
<area target="_blank" alt="checking default" href="https://www.hello.com/" shape="default">
</map>


html picture

=> the <picture> tag is used to define a container for multiple image resources.
the <source> tag is used to define multiple image resources for different display devices.
the <img> tag defined as the last element of the picture element as the backup if none of the above elements work and also we can style the width and height for all the images together at once only inside img tag.
Example : <picture style="float:left;margin-right:5px;">
<source srcset="../images/peakpx.jpg" media="(max-width: 600px)"> //0px< device-width <600px
<source srcset="../images/victory.jpg" media="(max-width: 900px)"> //600px < device-width < 900px
<img src="../images/workplace.jpg" alt="workplace" style="width:100%;height:200px" > //device-width > 900px
</picture>

OR

<picture>
<source srcset="../images/peakpx.jpg" media="(min-width: 900px)"> //device-width > 900px
<source srcset="../images/victory.jpg" media="(min-width: 600px)"> //600px < device-width < 900px
<img src="../images/workplace.jpg" alt="workplace" style="width:100%;height:200px" > //0px< device-width <600px

html tables

=>all the table content lie inside <table> tags.
the <td> tag defines a standard cell in an HTML table.
the <th> tag defines a header cell in an HTML table,basically it shows the cell content in bold.
the <tr> tag defines a row in an HTML table.
the <caption> tag defines a table caption, it should be the written as the first element inside the <table> tag.
the css is used to adjust the table properties like border, padding, width, height, etc.
