html tags and attributes

=>The tag name can either start from an alphabet or an underscore( _ ).
=> Attributes can provide additional information about the HTML elements on your page and control their behaviour.
Eg: <tag_name attribute_name="value"> Content Enclosed </tag_name>
=>HTML tags are not case sensitive, <p> means the same as <P>

html paragraphs

=>paragraphs are defined with the <p> tag, and is most commonly used for writing paragraphs
=>the user agent css style is : 

        p {
            display: block;
            margin-block-start: 1em;
            margin-block-end: 1em;
            margin-inline-start: 0px;
            margin-inline-end: 0px;
        }
=>eg. <p> paragraph </p>

=> the contents inside <pre> and </pre> tags are shown as it is, without any formatting. usually it reserves the space and the line breaks.
the user agent css style is: 
        pre {
            display: block;
            font-family: monospace;
            white-space: pre;
            margin: 1em 0px;
        }
=> the <br> tag is used to insert a single line break.
=> the <hr> tag is used to insert a horizontal line below. the user agent css style is: 
 
    hr {
        display: block;
        unicode-bidi: isolate;
        margin-block-start: 0.5em;
        margin-block-end: 0.5em;
        margin-inline-start: auto;
        margin-inline-end: auto;
        overflow: hidden;
        border-style: inset;
        border-width: 1px;
    }

html text formatting

=> the <abbr> tag is used to insert an abbreviation.
      eg. <abbr title="World Health Organization">WHO</abbr> ->WHO
=> the <address> tag is used to define the contact information of the author or owner of the document or article.
      eg. <address> 123, abc street, xyz city, 123456 </address> ->
123, abc street, xyz city, 123456
=> the <b> and <strong> tags are used to make bold text.
      eg. <b> bold or strong text </b> -> bold or strong text
=> the <bdo> tag is used to insert a text with a different direction.
      eg. <bdo dir="rtl"> text with a different direction </bdo> -> text with a different direction
=> the <blockquote> tag is used to insert a block of quoted text.
      eg. <blockquote> quoted text </blockquote> ->
quoted text
=> the <cite> tag is used to insert a citation.
      eg. <cite> citation </cite> -> citation
=> the <code> , <kbd> , <samp> tags are used to make a text look like written by typewritter.
      eg. <code> this is the way </code> -> this is the way
=> the <del> and <s> tags are used to make a deleted or strikethrough text.
      eg. <del> deleted or strikethrough text </del> -> deleted or strikethrough text
=> the <dfn> tag is used to insert a definition.
      eg. <dfn> definition </dfn> -> definition
=> the <em> and <i> tag is used to insert emphasized or italic text.
      eg. <em> emphasized or italic text </em> -> emphasized or italic text
=> the <ins> and <u> tags are used to insert a underlined or inserted text.
      eg. <u> underlined or inserted text </u> -> underlined or inserted text
=> the <mark> tag is used to insert a marked text.
      eg. <mark> marked text </mark> -> marked text
=> the <pre> tag is used to insert a preformatted text, the spaces and line breaks are kept as it is.
      eg. <pre> preformatted text </pre> ->
 preformatted text
=> the <small> tag is used to make small sized text.
      eg. <small> small text </small> -> small text
=> the <sub> tag is used to insert a subscript text.
      eg. (a + b)<sub> 2 </sub> ->(a + b)2
=> the <sup> tag is used to insert a superscript text.
      eg. (a + b)<sup> 2 </sup> ->(a + b)2
=> the <var> tag is used to insert a variable.
      eg. <var> variable </var> -> variable
=> the <wbr> tag is used to insert a possible line-break.
      eg. <wbr> possible line-break </wbr> -> possible line-break

html links

=> the <a> tag is used to insert a link.
      eg. <a href="http://www.google.com"> google </a> -> google
=> By default, links will appear as follows in all browsers:
>An unvisited link is underlined and blue
>A visited link is underlined and purple
>An active link is underlined and red
=> By default, the linked page will be displayed in the current browser window.
Use the attribute "target" to open the linked document in a new browser window or tab:
Example: <a href="https://www.w3schools.com/" target="_blank">Visit W3Schools!</a> -> Visit W3Schools!
=> to use image as link , we just have to put the img tag inside the <a> tag.
<a href="../learning_html/login_page.html"> <img src="../images/icons8-gunter-100.png" alt="penguin photo" target="_blank" style="width:42px;height:42px;"> </a> ⇛
=>Use mailto: inside the href attribute to create a link that opens the user's email program (to let them send a new email):
Example: <a href="mailto:udaybiswasofficial@gmail.com"> email </a> ->Send email to us.
=> if the page is too long , then to access different parts of the page :
First, use the id attribute to create a bookmark:
<h3 id="quote">html quotation</h3>
Then, add a link to the bookmark using a "#" before that id:
Example : <a href="#quote">html quotation</a> -> html quotation
You can also add a link to a bookmark on another page:
<a href="static_page.html#bold1"> players list </a> -> players list
=> the <link> tag is used to link external files.
      eg. <link rel="stylesheet" href="mystyle.css">
=> the <nav> tag is used to define navigation links.
      eg. <nav> navigation links </nav> ->
navigation links
